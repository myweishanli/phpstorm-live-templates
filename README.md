## PHP templates

### rmt
```php
$this->registerMetaTag([
    'name' => '$TAGNAME$',
    'content' => $SELECTION$,
]);
```
Yii2 | Register Meta-tags

### yiitrans
```php
$transaction = Yii::$app->db->beginTransaction();
try {
    $SELECTION$
} catch (\Exception $e) {
    $this->addError('exception', YII_DEBUG ? $e->getMessage() : Yii::t('common', 'Exception'));
}
if ($this->hasErrors()) {
    $transaction->rollBack();
} else {
    $transaction->commit();
}
```

Yii2 | DB transaction
